# Mac Admins Europe 2026

Official website for Mac Admins Europe 2026 conference, taking place on April 30th in Leiden, Netherlands.

A community-driven event for Apple administrators across Europe to share knowledge, network, and learn about the latest in macOS and iOS device management.

## Quick Start

```bash
npm install    # Install dependencies
npm run dev    # Start dev server
npm run build  # Build for production
```
