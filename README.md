# Mac Admins Europe 2026

[![Deploy to GitHub Pages](https://github.com/MacAdminsEurope/mac-admins-europe/actions/workflows/deploy.yml/badge.svg)](https://github.com/MacAdminsEurope/mac-admins-europe/actions/workflows/deploy.yml)

Official website for Mac Admins Europe 2026 conference, taking place on April 30th in Leiden, Netherlands.

A community-driven event for Apple administrators across Europe to share knowledge, network, and learn about the latest in macOS and iOS device management.

## Built with Astro

This site is built with [Astro](https://astro.build/), a modern static site generator that's been fantastic for creating fast, content-focused websites. Astro's zero-JavaScript-by-default approach means blazing-fast page loads, while still allowing interactive components when needed. Paired with GitHub Pages for hosting, it's been an incredibly smooth experience getting this site up and running.

## Hosted

This site is hosted by [GitHub Pages](https://pages.github.com/) via GitHub Actions.
